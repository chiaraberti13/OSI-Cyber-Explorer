## 🇬🇧 English

### Overview
**OSI Cyber Explorer** is a bilingual interactive platform designed for deep study of the OSI (Open Systems Interconnection) model and its security implications. The application combines visual packet encapsulation logic with technical deep-dives into layer-specific cybersecurity attacks and defenses. It is built as a hands-on laboratory for students and IT professionals to understand how data moves through a network and how each layer can be compromised or hardened.

### Educational Focus
- **Visual Encapsulation**: Understand how PDUs (Protocol Data Units) change from Layer 7 down to Layer 1.
- **Layer-Specific Threats**: Analysis of real-world attacks like SQLi (L7), SYN Flood (L4), and BGP Hijacking (L3).
- **Hardening Strategies**: Detailed mitigation roadmaps and countermeasures for each network layer.
- **Bilingual Interface**: Content available in both English and Italian to support global technical learning.

### Key Features
- **Interactive OSI Stack**: Dynamic visualization of the 7 layers of networking.
- **Protocol Analysis**: Deep-dives into core protocols (HTTP, DNS, SSH, TCP, IP, BGP).
- **Cybersecurity Academy**: Randomized quiz engine to verify networking and security knowledge.
- **Security Analysis Tab**: Expert technical insights and hardening roadmaps for each layer.

### Quick Start
#### Prerequisites
- Node.js 18+
- npm (included with Node.js)

#### Installation
```bash
npm install
```

#### Run in Development
```bash
npm run dev
```
App URL: `http://localhost:3000`

## 🇮🇹 Italiano

### Panoramica
**OSI Cyber Explorer** è una piattaforma interattiva bilingue progettata per lo studio approfondito del modello OSI (Open Systems Interconnection) e delle sue implicazioni di sicurezza. L'applicazione combina la logica visiva dell'incapsulamento dei pacchetti con analisi tecniche dettagliate di attacchi e difese specifici per ogni livello. È concepita come un laboratorio pratico per studenti e professionisti IT per capire come i dati si muovono attraverso una rete e come ogni livello possa essere compromesso o protetto.

### Focus Educativo
- **Incapsulamento Visivo**: Comprendere come le PDU (Protocol Data Units) cambiano dal Livello 7 fino al Livello 1.
- **Minacce per Livello**: Analisi di attacchi reali come SQLi (L7), SYN Flood (L4) e BGP Hijacking (L3).
- **Strategie di Hardening**: Roadmap dettagliate di mitigazione e contromisure per ogni livello di rete.
- **Interfaccia Bilingue**: Contenuti disponibili in inglese e italiano per supportare l'apprendimento tecnico globale.

### Funzionalità principali
- **Stack OSI Interattivo**: Visualizzazione dinamica dei 7 livelli del networking.
- **Analisi dei Protocolli**: Approfondimenti sui protocolli fondamentali (HTTP, DNS, SSH, TCP, IP, BGP).
- **Accademia Cybersecurity**: Motore quiz randomizzato per verificare le conoscenze di rete e sicurezza.
- **Tab Analisi di Sicurezza**: Insight tecnici esperti e roadmap di hardening per ogni livello.
